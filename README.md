# Gearovo Admin Dashboard

A clean, modern admin dashboard for managing orders, designed to match the [gearovo.com](https://gearovo.com) website aesthetic.

## Features

- **Orders List** – View all orders with filtering by status (All, processing, delivered, failed, pending)
- **Order Details** – View and edit customer info, status, tracking, payment, and items
- **Add Order Form** – Manually create new orders with full data entry

## Design

The dashboard uses the Gearovo brand colors:
- **Primary accent**: Dark teal (#00897B)
- **Backgrounds**: White and light gray
- **Typography**: Clean Inter sans-serif

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
```

## Tech Stack

- React 18
- React Router 6
- Vite
- Tailwind CSS
